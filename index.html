1ª Aula - Introdução ao HTML na Prática

Inspetor de Elementos

Ferramenta dos navegadores que permite visualizar e editar o HTML e CSS de qualquer site. Muito útil para estudo e testes.

Estrutura Básica do HTML

O HTML tem uma estrutura fixa: <!DOCTYPE html>, <html>, <head> e <body>. Dentro de <body> colocamos o conteúdo visível.

Falando Sobre Tags

As tags são os blocos de construção do HTML. Temos tags para títulos (<h1>), textos (<p>), imagens, listas e muito mais.

Atributos das Tags

São informações adicionais nas tags, como href, src, alt, etc. Servem para configurar o comportamento ou estilo.

Textos

Podemos usar tags como <strong>, <i>, <u>, <mark>, <del>, <small> e <abbr> para formatar o texto.
